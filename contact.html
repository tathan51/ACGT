Notre équipe est à votre disposition sur notre serveur Discord via les channels mis a dispositions

Pour les créations ⚙️

Les personnes souhaitant partager leurs créations peuvent soumettre un ticket sur notre serveur Discord. Une analyse de votre ticket sera effectuée dans les meilleurs délais.

📋 Ce qui est attendu dans votre ticket

Afin que notre équipe puisse vous répondre efficacement et rapidement, merci de préparer les informations suivantes lors de la rédaction de votre ticket pour partagez vos créations :

  • Votre pseudo : Précisez votre pseudo que vous souhaiteriez mettre en avant sur le site.
  • Nom du mods : Mettez nous le nom de votre création.
  • Description & caractéristiques de votre création : Expliquez-nous en détail ce que vous avez créé, les fonctionnalités qu'elle offre, caractéristiques techniques, ainsi que la version de votre mods.
  • Fichier : Fournissez le fichier de votre création (format accepté : .zip, .rar, etc.).
⚠️ Attention : Tout ticket incomplet ou irrespectueux prendra plus de temps à être traité, voire pourra être fermé sans suite.
